# Public Assets

Place your profile picture here with the filename: `profile.jpg`

## Profile Picture Requirements:
- **Filename**: `profile.jpg` (or `.png`, `.webp`)
- **Recommended size**: 400x400px or larger
- **Aspect ratio**: Square (1:1) works best
- **Format**: JPG, PNG, or WebP

The profile picture will be displayed in the Hero section with a circular frame and gradient glow effect.
